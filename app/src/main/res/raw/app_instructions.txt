NCM 转换器 — 使用说明
======================

一、应用简介
-----------
NCM 转换器是一款免费的 Android 工具，可将网易云音乐加密的 .ncm 文件解密还原为标准 MP3/FLAC 音频文件。

所有解密计算均在手机本地完成，不会上传您的任何文件到网络。

二、支持格式
------------
- 网易云音乐：.ncm

三、使用步骤
-----------
1. 点击「选择 NCM 文件」按钮，从文件管理器中选择一个或多个 .ncm 文件
2. 点击文件右侧的播放按钮开始解密，或点击底部的「全部转换」批量处理
3. 解密完成后点击保存按钮将文件写入存储（或等待自动保存）
4. 点击顶栏齿轮图标进入设置，可自定义导出路径、主题等

四、设置说明
-----------
- 保存路径：设置解密后的文件保存位置，默认为 Music/NCMConverter
- 手动点击保存：开启后需手动点击保存按钮；关闭后转换完成自动保存
- 主题模式：支持跟随系统 / 浅色 / 深色三种模式

五、歌词功能（需自行搭建 API 服务）
----------------------------------
歌词自动嵌入功能需要配合自建的网易云音乐 API 服务使用。

搭建参考项目：https://github.com/NeteaseCloudMusicApiEnhanced/api-enhanced

获取到API地址后请填入APP设置页面的API地址内。如通过Vercel部署，请通过代理访问或填入RealIP参数。

歌词设置说明：
- 自动嵌入歌词：开启后转换时自动搜索并嵌入歌词
- 歌词模式：
  · 混合模式：原文与翻译逐行合并显示
  · 原文：仅显示原文歌词
  · 翻译：仅显示翻译歌词
- API 地址：填入你部署的 API 服务地址（必填）
- Real IP：可选，部分 API 部署需要此参数

歌词嵌入格式：
- MP3 文件：写入 USLT（非同步歌词）ID3v2 标签
- FLAC 文件：写入 LYRICS Vorbis Comment 字段

六、技术细节
-----------
- 使用 AES-128-ECB 解密密钥块和元数据
- 使用改良 RC4 算法解密音频数据
- 自动检测输出格式（MP3/FLAC）
- 自动写入歌曲元数据（标题、艺术家、专辑、封面）
- 歌词获取支持多种匹配策略（musicId 直接查找、歌曲名+歌手搜索）

七、注意事项
-----------
- 每次最多可选择 50 个 .ncm 文件，导入时会自动过滤非 .ncm 文件
- 大文件解密可能需要较长时间，请耐心等待
- 解密后的音频格式由原始文件决定，通常为 MP3 或 FLAC
- 如果转换失败，请确认文件未损坏且为有效的 NCM 文件
- Android 10+ 使用 MediaStore API 保存文件
- 歌词获取需要网络连接，获取失败不影响音频文件转换

八、隐私声明
-----------
- 所有文件解密均在您设备本地完成
- 应用不会上传任何文件或音频数据到互联网
- 仅在获取封面图片和歌词时会访问网络
- 歌词获取通过用户自建的 API 服务进行
- 应用不收集任何个人信息

九、常见问题
-----------
Q: 为什么转换后的文件没有歌曲名？
A: 部分旧版本 .ncm 文件的元数据信息不完整，音频数据本身不受影响。

Q: 支持批量转换吗？
A: 支持。您可以选择最多 50 个文件后点击「全部转换」。

Q: 转换后的文件保存在哪里？
A: 默认保存在 Music/NCMConverter 目录下，可在设置中修改。

Q: 歌词功能无法使用？
A: 请确保已在设置中正确配置 API 地址，并且 API 服务正常运行。

Q: 为什么没有找到歌词？
A: 可能原因：1) API 服务未部署或地址错误 2) 歌曲在网易云音乐无歌词 3) 网络连接问题 4) 知らない
